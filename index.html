How it works

Three steps. No account required.

1

Paste the YouTube URL

Copy any YouTube video link and paste it into the input. Works with full URLs, short youtu.be links, and video IDs.

2

Click Get Transcript

The server fetches the caption track directly from YouTube's timedtext endpoint. No API key is used at any point.

3

Copy, download, or use it

One-click copy the full transcript with timestamps. Paste it into ChatGPT, save it as a file, or click any line to jump in the video.

FAQ

Common questions answered.

Is this free? ⌄
Yes, completely free. No account, no API key, no limits on the number of transcripts you generate.
Why can some videos not be transcribed? ⌄
Transcripts are only available when the video has captions — either auto-generated by YouTube or manually added by the uploader. Private videos and age-restricted content are also inaccessible.
How many channels can I track? ⌄
Up to 50 channels. Your channel list is stored in your browser's local storage — it persists across sessions and requires no account.
Does this use the YouTube Data API? ⌄
No. Channel video listings use YouTube's public Atom RSS feed (no API key required). Transcripts are fetched directly from YouTube's timedtext caption endpoint. No API quota is consumed.
How do I use this with ChatGPT? ⌄
Click Copy after generating a transcript, then paste into ChatGPT with a prompt like "Summarise this transcript" or "Extract the key points from this video."
